说明：

  1. 点击"智能推荐"按钮后，系统会调用后端 API 获取推荐
  2. 推荐成功后，会显示"智能推荐理由"面板
  3. 面板中显示每个推荐知识点的：
    • 红色：需要加强的薄弱点（正确率低）
    • 橙色：需要复习的知识点（长时间未练习）
    • 绿色：新的未练习知识点
    • 紫色：基础巩固知识点
  4. 每个知识点显示具体的推荐理由和相关数据
  5. 可以手动关闭面板，或等待自动隐藏
