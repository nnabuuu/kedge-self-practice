## 系统简介

智能自主练习系统是一款面向初中学生的在线学习平台，旨在提升学生自主学习能力，帮助教师实现精准教学，推动个性化教育的发展。

## 用户角色

| 角色 | 主要功能 |
|------|----------|
| **学生** | 自主练习、错题复习、学习分析、题目反馈 |
| **教师** | 题库管理、学生管理、数据分析、反馈处理 |
| **管理员** | 用户管理、系统配置、数据管理 |

## 快速开始

{% tabs %}
{% tab title="学生" %}
1. 使用学号和密码登录系统
2. 选择要练习的科目
3. 选择知识点范围
4. 配置练习参数（题型、数量等）
5. 开始答题练习
6. 查看练习结果和分析
{% endtab %}

{% tab title="教师" %}
1. 使用教师账号登录系统
2. 进入管理中心
3. 上传DOCX文档批量导入题目
4. 或手动创建题目
5. 查看学生练习数据
6. 处理学生反馈
{% endtab %}

{% tab title="管理员" %}
1. 使用管理员账号登录系统
2. 管理用户账号
3. 配置系统参数
4. 监控系统运行状态
{% endtab %}
{% endtabs %}
