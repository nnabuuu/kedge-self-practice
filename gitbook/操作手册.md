# 智能自主练习系统 - 操作手册

{% hint style="info" %}
**文档版本**: V1.0 | **最后更新**: 2025年1月 | **编制单位**: 上海即驰教育科技有限公司
{% endhint %}

本手册详细介绍智能自主练习系统的各项功能操作方法，帮助学生、教师和管理员快速上手使用系统。

---

## 学生功能

### 1. 账户管理

{% tabs %}
{% tab title="注册登录" %}
**登录系统**
1. 打开系统首页
2. 输入账号（学号）和密码
3. 点击"登录"按钮
4. 首次登录建议修改默认密码

**记住登录**
* 勾选"记住我"可保持7天登录状态
* 公共设备请勿勾选此选项
{% endtab %}

{% tab title="个人信息" %}
**查看个人信息**
* 点击右上角头像进入个人中心
* 可查看：姓名、班级、角色、注册时间

**修改密码**
1. 进入个人中心
2. 点击"修改密码"
3. 输入旧密码和新密码
4. 确认提交
{% endtab %}

{% tab title="偏好设置" %}
系统会自动保存您的练习偏好：
* 上次访问的科目
* 练习配置参数
* 界面显示偏好
{% endtab %}
{% endtabs %}

### 2. 科目选择

{% hint style="success" %}
**快速开始**：登录后在首页选择要练习的科目即可开始学习
{% endhint %}

**操作步骤**：
1. 在首页查看所有可用学科（如：历史、生物等）
2. 点击学科卡片进入该科目
3. 系统会记住您上次选择的科目，下次登录时快速访问

### 3. 知识点选择

系统采用树形结构组织知识点，方便您精准选择练习范围。

**知识点层级**：
```
📚 学科
  └── 📖 册 (Volume)
       └── 📑 单元 (Unit)
            └── 📄 课 (Lesson)
                 └── 📝 小节 (Section)
```

{% tabs %}
{% tab title="浏览知识点" %}
1. 进入科目后，左侧显示知识点树
2. 点击展开/收起各级节点
3. 查看每个知识点关联的题目数量
{% endtab %}

{% tab title="搜索知识点" %}
1. 在搜索框输入关键词
2. 系统实时显示匹配结果
3. 点击搜索结果直接定位
{% endtab %}

{% tab title="选择练习范围" %}
1. 勾选一个或多个知识点
2. 支持选择整个单元或具体小节
3. 选择后显示可练习题目总数
{% endtab %}
{% endtabs %}

### 4. 练习配置

在开始练习前，您可以自定义练习参数：

{% tabs %}
{% tab title="题目来源" %}
| 选项 | 说明 |
|------|------|
| **新题练习** | 只练习未做过的题目 |
| **包含错题** | 新题+之前做错的题目 |
| **仅错题练习** | 只练习做错的题目 |
{% endtab %}

{% tab title="题型过滤" %}
可选择练习的题型：
* ✅ 单选题 (single-choice)
* ✅ 多选题 (multiple-choice)
* ✅ 填空题 (fill-in-the-blank)
* ✅ 主观题/简答题 (subjective)
{% endtab %}

{% tab title="其他设置" %}
* **题目数量**：设置练习题数（或选择无限制）
* **时间限制**：可选设置练习时长
* **题目顺序**：顺序出题或随机打乱
* **显示解析**：答题后是否显示解析
* **自动跳转**：答对后自动跳转延迟时间
{% endtab %}
{% endtabs %}

### 5. 答题练习

{% hint style="warning" %}
**重要提示**：练习过程中系统会自动保存进度，中途退出不会丢失已答题目
{% endhint %}

#### 题目展示与作答

{% tabs %}
{% tab title="单选题" %}
* 点击选项即可选择答案
* 选择后自动提交（可在设置中关闭）
* 立即显示对错反馈
{% endtab %}

{% tab title="多选题" %}
* 点击勾选多个选项
* 点击"提交答案"确认
* 需要全部选对才算正确
{% endtab %}

{% tab title="填空题" %}
* 在输入框中填写答案
* 支持多个空格分别填写
* 系统支持备选答案判定
* 可申请AI重新评估
{% endtab %}

{% tab title="主观题" %}
* 在文本框输入答案
* 支持语音输入（点击麦克风图标）
* AI自动评分并给出评分理由
{% endtab %}
{% endtabs %}

#### 导航与控制

| 操作 | 方法 |
|------|------|
| 上一题 | 点击"上一题"按钮或按 `←` 键 |
| 下一题 | 点击"下一题"按钮或按 `→` 键 |
| 跳转题目 | 点击题号列表中的具体题号 |
| 暂停练习 | 点击"暂停"按钮 |
| 结束练习 | 点击"结束练习"按钮 |

### 9. 知识点分析

{% hint style="info" %}
知识点分析帮助您全面了解学习状况，找到需要加强的知识点
{% endhint %}

**功能说明**：
* **总体掌握度**：查看所有知识点的整体掌握情况
* **知识点热力图**：可视化展示各知识点的熟练程度
* **薄弱知识点**：列出需要加强的知识点（正确率<60%）
* **进步趋势**：查看学习进步曲线
* **强化训练**：一键开启薄弱知识点强化练习

### 10. 题目反馈

发现题目有问题？您可以向老师反馈：

{% tabs %}
{% tab title="提交反馈" %}
1. 在答题页面点击"报告问题"按钮
2. 选择问题类型：
   * 显示错误
   * 答案错误
   * 知识点关联错误
   * 题目重复
   * 表述不清

4. 提交反馈

### 3. DOCX文档解析工具

{% hint style="info" %}
这是教师快速创建题目的强大工具，支持从Word文档中自动提取和生成题目
{% endhint %}

**使用步骤**：

```
📄 上传文档 → 🤖 AI解析 → ✏️ 编辑调整 → 🏷️ 关联知识点 → 💾 批量保存
```

{% tabs %}
{% tab title="步骤1：上传文档" %}
1. 拖拽或点击上传Word文档
2. 文档应包含高亮标注的重要内容
3. 支持包含图片的文档
{% endtab %}

{% tab title="步骤2：配置解析" %}
* 选择LLM提供商（OpenAI/Azure）

* 设置生成题目数量
{% endtab %}

{% tab title="步骤3：AI解析" %}
* 点击"开始解析"
* 系统自动：
  * 提取高亮文本
  * 识别图片内容
  * 生成题目和答案
  * 推荐知识点关联
{% endtab %}

{% tab title="步骤4：编辑确认" %}
* 预览生成的题目
* 手动编辑和调整

* 完善解析说明
{% endtab %}

{% tab title="步骤5：保存导出" %}

## 常见问题

<details>
<summary><strong>Q: 忘记密码怎么办？</strong></summary>

请联系班主任或管理员重置密码。管理员可在"用户管理"中重置任意用户的密码。
</details>

<details>
<summary><strong>Q: 练习中途退出会丢失进度吗？</strong></summary>

不会。系统会自动保存练习进度，下次可以继续未完成的练习。
</details>

<details>
<summary><strong>Q: 填空题答案为什么判错了？</strong></summary>

填空题支持多种备选答案，但可能未覆盖所有正确表述。您可以点击"AI重新评估"，或提交反馈让老师审核。
</details>

<details>
<summary><strong>Q: 如何查看我的错题？</strong></summary>

进入"练习菜单"，选择"错题本练习"即可查看和练习所有错题。
</details>

<details>
<summary><strong>Q: 教师如何批量导入题目？</strong></summary>

有两种方式：
1. Excel导入：下载模板，填写后上传
2. DOCX导入：上传包含高亮内容的Word文档，AI自动解析生成题目
</details>
